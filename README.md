## 目的

平常在工作中、学习中写了很多东西,有的是业务、有的是模块、有的是知识点。只是不久之后写的那些东西在脑海里只有一些模糊的印象了
某个地方遇到,却发现不知道在哪里去找,代码写了一大堆,结果最后还是没有找到

知识没有沉淀,就是这个样子。所以做这个主要是想将之前的一些东西能整合到一起,哪怕整合不到一起,也至少对已用过的知识点有一些回顾

为了能比较详细的学习，一般都不使用默认配置
